# ServerHomeControl

Framework:    
-------
###### Beego ######  
start server:  
```bee run```

API Documentation:  
-------  
###### swagger ######  
```host:port/swagger```  
Build documentation:  
```bee run -downdoc=true -gendoc=true```

Build:  
-------  
###### bash-script ######  
build server to `/build/dist/`: `mac.zip, mac_x64.zip, linux.zip, linux_x64.zip, win.zip, win_x64.zip`  
Run file `bash_build_project`
